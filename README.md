# MovieSearchApp

네이버 영화검색 API를 활용한 iOS 어플리케이션.

## Description
- 최소 타겟 : iOS 14.0
- Portrait 모드만 지원
- MVVM 아키텍처 패턴 적용
- Storyboard를 활용하지 않고 코드로만 UI 구성
